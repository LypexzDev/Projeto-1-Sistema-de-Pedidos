1️⃣ Sistema de Pedidos
Descrição

Um sistema simples para gerenciar usuários, produtos e pedidos, salvando tudo em JSON. Permite cadastrar, listar e criar pedidos, calculando automaticamente o total.

Arquivos

usuarios.json – lista de usuários cadastrados

produtos.json – lista de produtos disponíveis

pedidos.json – lista de pedidos feitos

Funcionalidades principais

Cadastrar usuário: solicita nome e email, adiciona ao JSON com ID incremental.

Cadastrar produto: solicita nome e preço, adiciona ao JSON com ID incremental.

Criar pedido: solicita usuário e produtos por ID, soma valores e salva no JSON.

Listar dados: imprime usuários, produtos ou pedidos no console.

Lógica de programação

Uso de listas e dicionários para armazenar dados temporários

Funções para cada operação (cadastrar, listar, criar pedido)

Loop while para menu interativo
